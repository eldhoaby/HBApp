# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-05-15

### Added
- **Authentication**: JWT-based secure user authentication and authorization.
- **Role-Based Access Control**: Differentiated dashboards for Admin, Hotel Owners, and Users.
- **Room Management**: Hotel owners can add, edit, and manage their room inventory.
- **Booking System**: Real-time room availability checking and reservation logic.
- **Payment Integration**: Fully functional payment gateway integration (Stripe/Razorpay).
- **Responsive UI**: Built with React, Vite, and Tailwind CSS for mobile and desktop compatibility.
- **Professional Repository Structure**: CI/CD workflows, documentation, and polished root configuration.
